My name is Shaskia Putri Devi, in this project I created Foodly, a mobile "Food Delivery App" UI/UX design that focuses on providing a designed to create a simple, friendly, and visually comfortable user experience. The interface uses a green color theme to provide a fresh and eye-friendly look, helping users browse menus, order food, and track deliveries effortlessly.

In this project, I designed the complete user journey, starting from the welcome and authentication screens, exploring menus, placing orders, tracking delivery status, and viewing order history.

This project was designed using Figma, with the help of plugins such as Unsplash to provide high-quality images and create a more realistic interface.
